How to find Factorial of 5 in javascript?

Factorial is to multiply numbers ex: 5! = 5*4*3*2*1 = 125

                                 
                                    var factorial=(num)=>{
                    
                                        var res=1
                                    
                                        for(var i=num;i>1;i--){
                                            res = res*i
                                        }
                                        return res
                                    }
                                    num=5
                                    console.log(factorial(num))
